# 쿠팡 Growth — 센터분리 관리 탭 추가

## 범위

- 탭 1개 + 플레이스홀더 페이지 (기능·API·DB 없음)
- 탭 순서: **맨 뒤** (재고 현황 다음)
- 기존 [`layout.tsx`](src/app/(dashboard)/data/coupang-growth/layout.tsx)의 `PageTabsNav`는 [`coupangGrowthTabGroup`](src/config/page-tabs.ts)만 읽으므로 **탭 설정 1곳 수정**으로 반영됨

## 변경 파일

### 1. 탭 등록 — [`src/config/page-tabs.ts`](src/config/page-tabs.ts)

`coupangGrowthTabGroup.tabs` 배열 **끝**에 추가:

```ts
{
  title: "센터분리 관리",
  href: "/data/coupang-growth/center-separation",
},
```

최종 탭 순서:
1. 쿠팡 판매자 계정 관리
2. 재고 현황
3. **센터분리 관리** (신규)

### 2. 플레이스홀더 페이지 — 신규 [`src/app/(dashboard)/data/coupang-growth/center-separation/page.tsx`](src/app/(dashboard)/data/coupang-growth/center-separation/page.tsx)

[`excel-upload/page.tsx`](src/app/(dashboard)/sync/coupang-growth/excel-upload/page.tsx)와 동일한 페이지 헤더 패턴:

- `requireProfile()`
- `h1`: 센터분리 관리
- `p`: 센터별 입고·재고 분리 설정을 관리합니다. (준비 중 안내 문구)
- 별도 Panel 컴포넌트·서비스는 만들지 않음

```tsx
export default async function CenterSeparationPage() {
  await requireProfile();

  return (
    <div className="space-y-6">
      <div className="space-y-2">
        <h1 className="text-2xl font-semibold tracking-tight">센터분리 관리</h1>
        <p className="text-muted-foreground">
          센터별 입고·재고 분리 설정을 관리합니다. 기능은 준비 중입니다.
        </p>
      </div>
    </div>
  );
}
```

## 범위 외

- 사이드바 [`navigation.ts`](src/config/navigation.ts) — 이미 `/data/coupang-growth`로 진입, 변경 없음
- `page.tsx` redirect — 여전히 첫 탭(판매자 계정)으로 이동, 변경 없음
- DB 마이그레이션·API·컴포넌트

## 검증

1. `/data/coupang-growth` 접속 시 상단 탭 3개 표시, **센터분리 관리**가 맨 오른쪽
2. 탭 클릭 시 `/data/coupang-growth/center-separation` 플레이스홀더 표시
3. `npm run build` 통과
